# Task Management App

This is a simple task management web application built with Angular.

## Description
The application allows users to create, view, update, and delete tasks.  
Users can also filter tasks by status and store data in localStorage so the task list remains after refreshing the page.

## Features
- Add new task
- Delete task
- Update task status
- Filter tasks by status
- Save data with localStorage
- Responsive user interface

## Technologies Used
- Angular
- TypeScript
- HTML
- CSS
- localStorage

## Project Purpose
This project was developed to practice frontend development with Angular, including component-based architecture, form handling, CRUD operations, and local data storage.

## Main Functionalities
- Create a new task with title, description, and status
- Change task status between Todo, In Progress, and Done
- Delete tasks from the list
- Display all tasks in a clean and simple interface
- Keep task data after page refresh using localStorage
